I'm currently conducting research under Prof. Dominic Liao-McPherson at UBC, focused on developing real-time, adaptive control strategies for multi-agent systems. The work combines concepts from static equilibrium analysis, optimization, and game theory to simulate and solve competitive multi-agent interactions.

Project Overview

  • Built a symbolic solver using Python (Sympy) to compute static equilibria in constrained systems
  • Implemented a multi-agent simulator to find and analyze Nash equilibria
  • Applied KKT conditions to characterize optimal strategies under shared constraints
  • Validated models using known analytical solutions to ensure correctness

Tools & Skills

  • Python (Sympy, NumPy)
  • Optimization theory, game theory, and equilibrium modeling
  • GitHub for version control and structured development workflow

This project will be developed full-time during Summer 2025, with a final deliverable including a simulation framework, full technical report, and visualization of agent dynamics under varying constraints.
